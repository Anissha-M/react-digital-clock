# Digital Clock

A live digital clock built with React.

## Features
- Live time updated every second
- Shows current date
- Blue night mode design

## Built with
- React
- useState hook
- useEffect hook
- CSS

## Concepts used
- useState — stores current time
- useEffect — starts setInterval timer
- setInterval — updates time every 1 second
- clearInterval — cleans up timer on unmount
- toLocaleTimeString — formats time nicely

## How to run locally
npm install
npm start
